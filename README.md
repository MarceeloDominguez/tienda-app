## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/tienda-app.git
```

Go to the project directory

```bash
  cd tienda-app
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm start
```
